#### ATENTION ####
Speed Tester will be installed by default in /tmp/speed_tester/

1. Installing Speed Tester
	tar xfvz speed_tester.tgz
	cd speed_tester/
	./setup.sh install
	
2. Removing Speed Tester
	tar xfvz speed_tester.tgz
	cd speed_tester/
	./setup.sh clean

3. Using Speed Tester
	cd /tmp/speed_tester/
	./speed_check.sh
